# 前言

轻松使用 Hexo 建站

Hexo 是一个快速、简洁且高效的博客框架，许多人使用它来建立一个博客网站。但是由于 Hexo 官方网站的教程并没有涉及到主题配置及自定义，甚至有一些 Hexo 主题连官方文档都没有，导致使用者需要阅读那些网上良莠不齐的教程。而本文档将提供一站式的 Hexo 教程。
